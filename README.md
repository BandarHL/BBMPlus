# BBMPlus open source

# Features:
- Unlimited Pings
- Download From Feeds
- Hide Read Status
- Hide Typing
- Hide Screen shot
- Hide Voice Call Button
- Hide Video Call Button
- Timed Messages Forever
- Confirm Voice Call and Video Call
- Hide StatusBar
- Disable Ads
- Full Screen For iPAD and iPhone X
